# :whale: LaTex 处理 PDF 文件 :whale:


## 1、LaTex 合并 PDF 文件

编译环境为 TeXstudio 2.12.10，OS 为 Win 10，使用以下代码。

```
\documentclass[a4paper]{article}
\usepackage{pdfpages}

\begin{document}
 
\includepdfmerge{m1.pdf,1-2} % 第一至第二页
\includepdfmerge{m2.pdf,1} % 第一页

\end{document}
```
